# Twitter Clone

> This is a simple twitter clone built in Rails. It has authentication and tweet ownership.

![screenshot](public/screenshot.png)

## Features

- Publish, edit and delete Tweets.
- Sign up, edit and delete user profiles.

## Built With

- Ruby 2.7.0
- Rails 6.0.2.2

Gems:
- Simple Form
- Devise
- Gravatar
- Bulma
